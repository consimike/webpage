UFC Prediction

I worked with two other individuals to create a machine learning model which could be used to predict UFC fights. Our first step was to find data, which we found the UFC fight history on kaggle.com and scraped UFC fighter data using the python library beautiful soup. After finding the raw data, we needed to clean it. The cleaning process was done using the python library pandas, and after the data was pristine condition we researched and determined the best machine learning model would be a neural network. We tweaked and tweaked the model until we came up with 1 input layer, 1 hidden layer, and 1 output layer. To create the webpage we used flask for our backend and Html/CSS and a bit of Javascript for the frontend.

At the end, and after much deliberation we achieved an accuracy of 71%.
